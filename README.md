#  AI 店铺 · 演示 Demo by chutian

## 部署到 GitHub Pages（5分钟）

### 步骤

1. **新建仓库**
   - 登录 GitHub（chutian1303-bot）
   - 点击右上角 `+` → `New repository`
   - 仓库名填：`ai-demo`
   - 选 `Public`，点击 `Create repository`

2. **上传文件**
   - 进入新建的仓库
   - 点击 `Add file` → `Upload files`
   - 把 `index.html` 拖进去
   - 点击 `Commit changes`

3. **开启 GitHub Pages**
   - 进入仓库 `Settings`
   - 左侧找 `Pages`
   - Source 选 `Deploy from a branch`
   - Branch 选 `main`，目录选 `/ (root)`
   - 点击 `Save`

4. **等待 1-2 分钟后访问**
   - 地址：`https://chutian1303-bot.github.io/jnby-ai-demo/`

## 体验路径建议

给客户演示时，可以按这个顺序引导：

1. **进店** → 看 anna 的欢迎语 + 新品推荐
2. **向下滑动** → 体验店招折叠、吸顶效果
3. **点击商品** → 查看详情浮层（推荐理由、尺码、搭配）
4. **在输入框提问** → 试试这几句话：
   - "帮我找适合通勤的外套"
   - "有什么优惠"
   - "给我推荐一套穿搭"
5. **点击「商品足迹」** → 查看浏览过的商品
6. **点击「历史消息」** → 查看问过的所有问题
